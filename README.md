# Evaluación Parcial de Scripting - Primer Parcial

Este repositorio contiene el código y los recursos relacionados con la primera evaluación parcial del curso de Scripting. Los participantes en esta evaluación son:

- Cristian Usuga (ID: 467982)
- Juan Guillermo Caicedo (ID: 464690)

## Descripción

En esta evaluación parcial, hemos trabajado en conceptos y aplicaciones relacionados con la estructura de datos de colas (`Queue`) en el contexto de programación. Hemos abordado la clasificación de tickets y la gestión de colas en un centro de atención al cliente.

## Contenido del Repositorio

- **Código Fuente:** Contiene el código fuente de las funciones y estructuras desarrolladas para la evaluación. Se han implementado soluciones para clasificar tickets y gestionar colas según los trámites requeridos.

- **Pruebas Unitarias:** Incluye pruebas unitarias para validar el funcionamiento correcto de las funciones implementadas. Estas pruebas aseguran que el código cumple con los requisitos y produce los resultados esperados.

- **Documentación:** Se proporciona documentación relevante para entender el enfoque, el diseño y la lógica detrás de las soluciones implementadas. Esto incluye explicaciones de las funciones clave y los enfoques utilizados.

## Instrucciones de Uso

1. Clone este repositorio en su sistema local.

2. Explore el código fuente en la carpeta correspondiente. Revise los archivos relacionados con la clasificación de tickets y la gestión de colas.

3. Ejecute las pruebas unitarias para verificar el correcto funcionamiento del código. Las pruebas ayudarán a asegurarse de que las funciones cumplan con los requisitos establecidos.

4. Consulte la documentación proporcionada para obtener más información sobre cómo se han abordado los desafíos y cómo funcionan las soluciones implementadas.
